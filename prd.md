# 제품 요구사항 정의서 (PRD)
## 프로젝트명: 암호화폐 실시간 시세 대시보드 (Crypto Real-time Dashboard)

---

## 1. 문서 개요
본 문서는 암호화폐 투자자를 위한 실시간 시세 및 시장 분석 대시보드 웹 서비스의 제품 요구사항을 정의합니다. 개발팀, 디자이너, 기획자 간의 원활한 소통을 위해 작성되었으며, 초보자도 개발 목적과 세부 기능을 쉽게 이해할 수 있도록 구성했습니다.

---

## 2. 제품 목표 및 배경
*   **배경**: 암호화폐 시장은 24시간 내내 실시간으로 변동하므로, 투자자들은 신속하고 직관적으로 시장 흐름을 파악해야 합니다. 그러나 초보 투자자들은 여러 화면을 돌아다니며 정보를 조합하는 데 어려움을 겪습니다.
*   **목적**: 시가총액 상위 100대 암호화폐의 실시간 가격 정보, 변동률, 시장 전반의 분위기 분석 결과, 그리고 시각화된 차트를 **한눈에 볼 수 있는 싱글 페이지 대시보드**를 제공합니다.

---

## 3. 대상 사용자 및 페르소나
*   **대상 사용자**: 국내외 암호화폐 시장의 흐름을 빠르고 간편하게 파악하고자 하는 개인 투자자.
*   **주요 요구사항**:
    *   현재 가격이 원화(KRW) 기준으로 정확하게 표시되어야 함.
    *   오늘 가장 많이 오른 코인과 떨어진 코인을 빠르게 알고 싶음.
    *   지난 7일간 가격이 어떻게 변화했는지 복잡한 분석 도구 없이 차트로 간편하게 확인하고 싶음.

---

## 4. 제품 핵심 가치 및 방향성
1.  **눈이 편안한 다크 모드**: 주야간 상시 모니터링을 위해 모던하고 몰입감 있는 **어두운 테마(Dark Mode)** 기반의 프리미엄 UI 디자인을 적용합니다.
2.  **데이터의 안정적 갱신**: 외부 API 호출량 제한을 고려하여 시스템을 설계하되, 사용자가 원할 때 즉시 갱신할 수 있는 유연성을 제공합니다.
3.  **직관적인 분석 정보**: 복잡한 지표 분석 없이도 시장이 현재 상승세인지 하락세인지 직관적으로 보여줍니다.

---

## 5. 기능적 요구사항 (Functional Requirements)

### [기능 1] 암호화폐 실시간 시세 조회 및 목록 제공
*   **상세 설명**: 시가총액 기준 상위 100개 코인의 상세 정보를 테이블(표) 형태로 출력합니다.
*   **화면 표시 정보**:
    *   **코인 정보**: 코인 로고 이미지, 국문/영문 코인명, 심볼(예: BTC, ETH)
    *   **현재 가격**: 대한민국 원화(KRW) 기준 (천 단위 쉼표 포맷팅 적용)
    *   **등락률**: 1시간전, 24시간(1일)전, 7일전 대비 등락률(%) 표시
        *   상승인 경우 빨간색(또는 초록색 글로벌 표준 선택 가능, 국내 기준은 주로 빨간색 상승/파란색 하락)과 `▲` 기호 표시
        *   하락인 경우 파란색과 `▼` 기호 표시
    *   **시가총액**: 원화(KRW) 기준 표시 (조, 억 단위 등 가독성 높은 포맷 적용 권장)
    *   **7일 가격 추이 차트**: 간결하면서도 상호작용이 가능한 선형 차트

### [기능 2] 상호작용형 7일 가격 추이 차트
*   **상세 설명**: 코인 목록 우측에 각 코인의 최근 7일간의 가격 흐름을 한눈에 볼 수 있는 미니 차트를 배치합니다.
*   **동작 및 스펙**:
    *   기본 상태에서는 심플한 선 그래프로 표시됩니다.
    *   사용자가 차트 영역에 마우스를 올릴(Hover) 경우, 특정 시점의 상세 가격과 날짜가 툴팁(Tooltip) 형태로 실시간 표시되는 **상호작용(Interactive) 기능**을 지원합니다.

### [기능 3] 코인 검색 및 정렬
*   **코인 검색**:
    *   검색창에 코인명(한글/영어) 또는 심볼명(BTC, ETH 등)을 입력하면 목록이 실시간으로 필터링되어 노출됩니다.
    *   일치하는 결과가 없을 시 "검색 결과가 없습니다" 메시지를 노출합니다.
*   **목록 정렬**:
    *   기본 정렬: 시가총액 순
    *   정렬 필터: 현재 가격순(높은순/낮은순), 24시간 등락률순(높은순/낮은순) 정렬 기능을 제공합니다.

### [기능 4] 시장 분석 및 현황 요약
*   **상승장 / 하락장 판별**:
    *   **판단 알고리즘**: 시가총액 상위 10개 코인의 24시간 평균 등락률을 계산합니다.
    *   평균 등락률이 **0% 이상(양수)**이면 화면에 "**상승장** (Bull Market)" 요약 카드를 노출하고 녹색/빨간색 불빛의 시각 효과를 줍니다.
    *   평균 등락률이 **0% 미만(음수)**이면 화면에 "**하락장** (Bear Market)" 요약 카드를 노출하고 파란색/회색 계열의 시각 효과를 줍니다.
*   **오늘의 상승/하락 코인 Top 5**:
    *   불러온 100개 코인 중 24시간 등락률이 **가장 높은 5개 코인(상승 Top 5)**과 **가장 낮은 5개 코인(하락 Top 5)**을 각각 카드 뉴스나 리스트 형태로 화면 상단에 요약하여 보여줍니다.

### [기능 5] 데이터 동기화 및 API 정책
*   **자동 동기화**: 대시보드가 열려 있는 동안 **5분마다** 백그라운드에서 API를 호출하여 시세 데이터를 최신화합니다.
*   **수동 새로고침**: 화면 우측 상단에 "새로고침" 아이콘/버튼을 제공하여, 사용자가 원할 때 클릭 시 즉시 최신 데이터를 불러옵니다.
*   **로딩 및 피드백**: 데이터를 불러오는 동안 화면에 로딩 애니메이션(스피너 등)을 표시하고, 수동 새로고침의 경우 남은 쿨타임(예: API 제한 방지를 위한 재요청 방지 5초)을 시각적으로 나타냅니다.

## 6. 기술 요구사항 & 제약조건 (Technical & Non-Functional Requirements)

### 6.1 API 스펙
*   **제공사**: CoinGecko Demo API
*   **연동 Endpoint**:
    *   `GET https://api.coingecko.com/api/v3/coins/markets`
    *   필수 Query Parameters:
        *   `vs_currency=krw` (원화 기준 가격)
        *   `order=market_cap_desc` (시가총액 순 정렬)
        *   `per_page=100` (상위 100개 데이터)
        *   `page=1` (첫 페이지)
        *   `sparkline=true` (7일 가격 추이 데이터를 포함하기 위해 필수 설정)
        *   `price_change_percentage=1h,24h,7d` (등락률 기간 설정)
*   **제약사항 관리**: Demo API Key의 호출 제한(Rate Limit)을 모니터링하여 예외 처리를 구현해야 합니다. (API 호출 실패 시 사용자에게 알림을 표시하고 직전 저장된 로컬 캐시 데이터를 유지하도록 처리)

### 6.2 개발환경 및 보안
*   **API Key 보안**: API Key는 소스코드 내부(클라이언트 스크립트 등)에 하드코딩하지 않습니다.
    *   `.env` 등 환경변수 설정 파일로 분리하여 관리합니다.
    *   배포 환경(Vercel, Netlify 등) 시스템 설정에 환경변수로 등록하여 주입받도록 구성합니다.
*   **기술 스택 추천**:
    *   **Frontend**: React 또는 Next.js (사용자와의 빠른 상호작용 구현 용이)
    *   **Chart Library**: Recharts 또는 Chart.js (Interactive sparkline 차트 구현용)
    *   **Styling**: Vanilla CSS 또는 TailwindCSS (어두운 색상의 모던한 스타일링)

### 6.3 UX/UI 가이드라인
*   **기본 테마**: 다크 모드 (배경: `#0f172a` / `#1e293b` 계열의 딥 블루 및 다크 그레이 계열 사용)
*   **시각화 피드백**: 숫자 증가/감소 시 깜빡임 애니메이션(Flash Effect)을 가볍게 주어 실시간 느낌을 배가시킵니다.
*   **반응형 레이아웃**: 모바일(가로 스크롤 테이블 또는 카드 전환), 태블릿, 데스크톱 화면 크기에 최적화된 반응형 Grid를 제공합니다.

## 7. 예외 처리 (Exception Handling)
1.  **API 호출 실패 (Network Error / Rate Limit 초과)**:
    *   "현재 데이터 연결이 원활하지 않습니다. 마지막으로 업데이트된 시각: XX:XX" 메시지와 함께 이전 캐시 데이터를 유지하여 화면이 비어 보이지 않게 처리합니다.
2.  **검색 결과 없음**:
    *   목록 전체가 사라지는 대신 "일치하는 코인이 없습니다" 안내 배너를 출력합니다.
3.  **차트 데이터 누락**:
    *   특정 코인의 sparkline 데이터가 불완전하거나 제공되지 않을 경우, 빈 선이나 "데이터 준비 중" 텍스트를 대체 노출합니다.

## 8. 향후 고도화 계획 (Roadmap)
*   **관심 코인 북마크**: 개별 투자자가 자주 모니터링하는 코인만 필터링해서 볼 수 있는 즐겨찾기(관심등록) 기능 추가.
*   **가격 알림 설정**: 특정 코인의 가격이 설정한 목표가에 도달할 시 브라우저 푸시 알림(Web Push Notification) 발송 기능.
*   **다양한 화폐 단위 지원**: 달러(USD), 유로(EUR), 엔화(JPY) 등 다국어 화폐 변환 기능 적용.
